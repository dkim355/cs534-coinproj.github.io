Motivation

For our research project, we will be creating a program that enumerates the value of coins spread out on a flat surface. To do this, it must recognize and separate each coin, then assign its respective value and add them together. We chose this project because counting coins by hand is a slow, tedious process, so we aspire to create a more efficient and accurate system for counting change using a computerâ€™s image processing abilities. Using grayscale, centroid, edge detection, and character recognition, we will identify each coin in an image and count the number of coins in the image. Moreover, currency detection and recognition will be possible and more accurate if we can collect enough images and information to build a database for comparison with the image in question.
