Motivation

Coin detection is one of many image recognition techniques used for image recognition-based computation. Currently, banks count coins with a machine that uses a weight or size system. Like the bankâ€™s coin counting machine, vending machines also use a weight or size system to distinguish different coin types. There has been several attempts and implementations to build a system for counting and recognizing coins.

However, there are several limitations such as photo angle, stacked coins, detecting and categorizing different types of coins while stacked, and coin proportion comparison. This program was selected for its challenging problems in its implementation and categorization of different types of coins.
